# patchup-website
PatchUp! merupakan usaha kreatif di bidang fashion handmade yang memproduksi tas, dompet, dan aksesori berbahan dasar kain perca (patchwork) dengan mengusung konsep upcycle. Melalui pemanfaatan limbah tekstil, PatchUp! mengolah potongan kain sisa menjadi produk baru yang memiliki nilai guna sekaligus nilai estetika.
